Defining an application from components

Source: concert-2.2.x-documentation.md (concert) One method for defining your applications is to select relevant images and repositories from your component library. The component library is populated based on existing data ingestion jobs or SBOM files that pull application component and lifecycle data from your third-party tools and services. Defining an environment from resources One method for defining your environments is to select relevant build artifacts from resources. The resource library is populated based on existing data ingestion jobs that pull application and environment data from your third-party tools and services.
