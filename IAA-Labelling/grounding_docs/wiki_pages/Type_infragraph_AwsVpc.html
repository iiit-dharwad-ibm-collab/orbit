AWS Virtual Private Cloud

Source: redp5671.md (turbonomic) The Amazon VPC service provisions a private, isolated section of the cloud where you can start AWS services and other resources in a vNet that you define. You have complete control over your virtual networking environment, including selection of an IP address range, creation of subnets, and configuration of route tables and network gateways. IBM Spectrum Virtualize for Public Cloud installation template in AWS Marketplace supports both creating a VPC or using an existing VPC.
