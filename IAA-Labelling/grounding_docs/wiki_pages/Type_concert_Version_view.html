In the Workflow version field, select the version of the workflow that you want to use when the job runs.

Source: concert-2.2.x-documentation.md (concert) To always use the latest saved version, select Always use latest version. To always use the version that is the latest at the time that the job is created, select Use current version. If you select this option, the job always runs this version, even if later versions are created. To use a specific version, select Use specific version, then select your preferred version in the Select specific version field.
