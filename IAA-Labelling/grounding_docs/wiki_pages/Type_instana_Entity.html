Entity analysis

Source: instana-observability-1.0.312-documentation.md (instana) The agent runs a tool call to collect data about an entity of interest and the entities around it. A comprehensive analysis across six entity categories (focal entity, upstream/downstream services, infrastructure, and related components) is then completed through the collection of trace logs, error messages, events, and metrics for each entity.
