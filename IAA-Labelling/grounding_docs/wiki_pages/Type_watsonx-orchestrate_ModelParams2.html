Overview

ModelParams2 type from watsonx-orchestrate.

Fields

FieldData TypeListDescription
decoding_methodStringRepresents the strategy that is used for picking the tokens during generation of the output text. Possible values include 'greedy' for deterministic generation and 'sample' for stochastic generation.
max_new_tokensIntThe maximum number of new tokens to be generated. It limits the length of the generated output, preventing it from exceeding a specified size.
min_new_tokensIntIf stop sequences are given, they are ignored until minimum tokens are generated.
random_seedIntRandom number generator seed to use in sampling mode for experimental repeatability.
repetition_penaltyIntRepresents the penalty for penalizing tokens that have already been generated or belong to the context. The value 1.0 means that there is no penalty.
return_optionsReturnOptions1
temperatureFloat
top_kIntThe number of highest probability vocabulary tokens to keep for top-k-filtering. Only applies for sampling mode. When decoding_strategy is set to sample, only the top_k most likely tokens are considered as candidates for the next generated token.
top_pIntSimilar to top_k except the candidates to generate the next token are the most likely tokens with probabilities that add up to at least top_p. Also known as nucleus sampling. A value of 1.0 is equivalent to disabled.
