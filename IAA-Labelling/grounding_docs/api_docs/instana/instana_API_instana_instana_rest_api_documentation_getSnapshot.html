Get snapshot details

GET

/api/infrastructure-monitoring/snapshots/{id}

Get all snapshot information

Request URL
https://{unit}-{tenant}.instana.io/api/infrastructure-monitoring/snapshots/{id}?to=REPLACE_THIS_VALUE&windowSize=REPLACE_THIS_VALUE
Copy to clipboard
https://{domain}/api/infrastructure-monitoring/snapshots/{id}?to=REPLACE_THIS_VALUE&windowSize=REPLACE_THIS_VALUE
Copy to clipboard
Security
apiKey
authorization
required in header

In order to use the REST API, you need to create an API token and assign permissions to it. API tokens are managed only by the account owner role, who can create multiple API tokens with varying permissions. API token management can found in Settings → Access Control → API Tokens.

Parameters
Name
Description
  • id
    REQUIRED
    string
    (path)

    snapshot id

  • to
    integer
     (int64)
    (query)

    end of timeframe expressed as the Unix epoch time in milliseconds

    Example: 1689018652000
  • windowSize
    integer
     (int64)
    (query)
